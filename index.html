API info
  • Creating new user: POST JSON { username : [your username] } to /api/exercise/new-user
  • Getting all users: GET to /api/exercise/users
  • Adding exercises: POST JSON { _id, description: String, duration: stringified Number, date (optional): String in a format YYYY-MM-DD } to /api/exercise/add
